# Wiring Guide

![Rotary Keyboard wiring diagram](../images/wiring-diagram.png)

## LCD

| LCD Pin | Arduino Nano Pin |
| --- | --- |
| VCC | 5V |
| GND | GND |
| SDA | A4 |
| SCL | A5 |

## Rotary Encoder

| Encoder Pin | Arduino Nano Pin |
| --- | --- |
| CLK / A | A0 |
| DT / B | A1 |
| SW | A2 |
| + | 5V |
| GND | GND |

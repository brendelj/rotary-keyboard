# Customization

## Keyboard Characters

Edit `kbChars` in `sketch.ino` to add or remove characters.

## LCD Size

The project currently targets a 20x4 LCD at I2C address `0x27`.

For a different LCD size, update the display setup and menu drawing code.

## Menu Items

To add a menu item, update `menuItems`, `menuLength`, and `menuSelect()`.

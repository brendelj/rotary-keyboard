# Contributing

Thanks for helping improve Rotary Keyboard.

## How to Contribute

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Test the sketch in Arduino IDE or Wokwi.
5. Open a pull request with a clear description.

## Coding Style

- Keep Arduino code readable and beginner friendly.
- Prefer clear names over short names.
- Document pin changes in `README.md` and `docs/pinout.md`.
- Keep examples small enough for Arduino Nano unless specifically targeting another board.
